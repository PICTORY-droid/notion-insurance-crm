# \# 🏦 보험설계사 TOP 1% CRM

# 

# 보험설계사를 위한 노션 올인원 CRM 자동화 코드입니다.

# 

# \## 📋 포함된 기능

# \- 👥 고객 DB — 고객 등급 자동 분류 (VIP/핵심/잠재/신규리드)

# \- 🔥 영업 파이프라인 — 칸반 보드로 영업 단계 관리

# \- 💬 상담 DB — 상담 내용 체계적 기록

# \- 📋 계약 DB — 계약 현황 및 만기일 자동 계산

# \- 📝 활동 DB — 전화/방문/문자/미팅 기록

# \- ⏰ 리마인드 DB — 오늘 할 일 자동 필터

# 

# \## 📁 파일 구조

# \- `setup-crm.js` — DB ID 목록

# \- `add-sample-data.js` — 샘플 데이터 추가

# \- `check-all.js` — 전체 DB 속성 점검

# \- `link-data-v2.js` — 샘플 데이터 릴레이션 연결

# 

# \## 🚀 시작하는 방법

# 1\. `.env` 파일에 NOTION\_TOKEN과 NOTION\_PAGE\_ID 입력

# 2\. `npm install`

# 3\. `node check-all.js` 로 DB 상태 확인
