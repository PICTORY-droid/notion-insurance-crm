# 보험설계사 TOP 1% CRM

보험설계사를 위한 노션 올인원 CRM 자동화 코드입니다.

## 포함된 파일
- setup-crm.js: DB 6개 생성
- add-sample-data.js: 샘플 데이터 추가
- check-all.js: 전체 DB 속성 점검

## 노션 페이지
https://phrygian-building-31e.notion.site/TOP-1-CRM-342d597ef70780998ab6e9c3adfcaa37

## 사용 방법
1. .env 파일에 NOTION_TOKEN과 NOTION_PAGE_ID 입력
2. npm install
3. node 파일명.js 실행
